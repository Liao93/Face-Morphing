# Face-Morphing
Image morphing through feature lines.

## 使用套件
- Python 3.7.1
- pyqt 5.9.2
- opencv 3.4.1
- numpy 1.19.2

## 使用方式
1. 使用 run.bat 啟動程式。
2. 在圖片上用滑鼠拖拉標記控制線(注意線的順序與方向要一致)。
3. 在畫面上可以選擇想要的 t 值，按 ”Start” 鈕會顯示結果，有三個視窗，分別是圖一、圖二的變形結果，跟顏色混合後的結果。
4. 按 ”Reset” 鈕可以清除控制線。
5. 按 “Animation” 鈕顯示新視窗(需要時間運算)，滑鼠連點視窗會播放變形動畫。
